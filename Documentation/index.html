Requirements

To install and run the Metrox Material React project properly, ensure your system meets the following requirements. You'll need to have Node.js installed on your machine, along with either npm (included with Node.js) or Yarn as your package manager.

You can download Node.js from the official website: Node.js. npm is included with Node.js installation. If you prefer Yarn, you can install it via npm with:

npm install -g yarn

Recommended Versions

  • Node.js: v18.x or higher
  • npm: v9.x or higher (included with Node.js)
  • Yarn: v4.x (optional - used in this project)

Key Dependencies

  • React: 19.2.0
  • Material-UI (MUI): 7.3.4
  • React Router: 7.9.3
  • Vite: 7.1.9 (Build tool)
  • ApexCharts: 5.3.5 (Charts library)
  • Framer Motion: 12.23.22 (Animations)
  • SWR: 2.3.6 (Data fetching)

Install Dependencies

To install all required packages, run one of the following commands at the root of the project:

Using Yarn (Recommended):

yarn install

Using npm:

npm install

Development Server

To start the development server, use one of the commands below. This will launch the app using Vite and serve it locally (typically on http://localhost:5173).

Using Yarn:

yarn start

Using npm:

npm run start

Build

To build the project for production, run:

Using Yarn:

yarn build

Using npm:

npm run build

React Installation

Local Development Setup

Follow these steps to set up the project locally:

  1. Extract the template files to your desired directory
  2. Open terminal/command prompt and navigate to the project directory
  3. Install dependencies: yarn install or yarn install
  4. Start the development server: yarn start or yarn start
  5. Open your browser and visit: http://localhost:5173

Installation into Hosting Server

Static Hosting Deployment

For static hosting services like Netlify, Vercel, or GitHub Pages:

  1. Build the project: yarn build or npm run build
  2. Upload the dist/ folder to your hosting service
  3. Configure your hosting to serve the index.html for all routes (SPA configuration)

Server Hosting Deployment

For traditional web servers:

  1. Build the project: yarn build or npm run build
  2. Upload the contents of the dist/ folder to your web server's public directory
  3. Configure your web server to rewrite all requests to index.html

Popular Hosting Platforms

Netlify

  1. Connect your repository to Netlify
  2. Set build command: yarn build or npm run build
  3. Set publish directory: dist
  4. Deploy automatically

Vercel

  1. Import your project from GitHub/GitLab
  2. Vercel will automatically detect Vite/React
  3. Deploy with zero configuration

GitHub Pages

  1. Build the project: yarn build or npm run build
  2. Commit and push the dist folder to your repository
  3. Enable GitHub Pages in repository settings
  4. Select the dist folder as source
